前言（一些废话）

今天来毕设实验室，准备挪一下拓展坞的位置，结果一个不小心 用力过猛，把拓展坞给拆了

其实本人对这个拓展坞不满已久——装着四个通常用于标识USB3.0的蓝色母座，却只能提供2.0的速度，蒙骗了不知多少实验室里的年轻后辈。（虽然说是我自己捡的，但是每次前后辈来我这参观，看见这明艳的蓝色母座，我只能讪讪然承认这只有USB2.0的速度，就有那么一丢丢尴尬）
再也是我对这小玩意的内部构造是个咋样一直很好奇，索性将错就错，直接把拓展坞里面的电路板拿出来看看。


一直以为内部电路板构造可能会很复杂的，但是拿到才发现只有一块HS8836A芯片、几个电阻电容和四个USB插口，大跌眼镜了家人们。
关于 HS8836A 芯片的资料在CSDN上没有相关博客介绍，本人就借此机会写下我的第一篇博客叭！

关于HS8836A芯片

HS8836A 是华声讯达电子公司推出的USB 2.0高速HUB控制器芯片，内嵌一个类RISC处理器，用来操作内部控制/状态寄存器并且响应USB主机的命令。

HS8836A设计上最大限度减少外围元件及BOM成本，并优化引脚布局支持单面和无跨线布版，有效的降低了方案的生产成本。

功能特点

  1. 完全兼容USB2.0标准，向下兼容USB1.1标准；
  2. USB2.0 HUB 主控 IC, 免驱, 免晶体；
  3. 支持封装：SSOP16、SOP16、SSOP24；
  4. 符合总线电源要求；
  5. 提供包含个体和整体的电源管理模式，提供了下行端口过流保护功能；
  6. 能够在自供电模式和总线供电模式上自动切换；
  7. 提供5V至3.3V工作电压；

HS8836A封装管脚

管脚定义

HS8836A芯片提供了一路USB输入（10脚、11脚）和 四路USB2.0输出（1、2脚，3、4脚，5、6脚，7、8脚），可以很好的满足USB2.0的拓展需求。

电路图

四路USB2.0输出的供电电压均来自于VBUS总线，同时芯片也通过VBUS供电，9脚REF端必须通过10kΩ的电阻与模拟信号的地（AGND）连接起来。12脚和14脚通过接地的滤波电容连接起来，拓展坞总体电路通过VBUS（即主机端电源）供电。

纯硬件设备是指内部不包含微处理器，无需烧写软件就能够运行的电子设备。如天线、老式收音机、老式电视机、老式洗衣机等。这类设备通常功能简单，易于操作，用户通常只需要打开电源开关并调节电位器、可变电容就能够使设备进行工作。嵌入式设备是指内部包含微处理器，需要烧写软件才能够运行的电子设备。由于具备微处理器和程序代码，因此嵌入式设备能够完成比较复杂的功能。用户通常需要按照屏幕上的菜单进行选择，才能够使设备工作于希望的模式。用户甚至可以对设备进行编程，使设备按照特定的时序进行工作。
